# User Profile

## この利用者について

この repository の利用者は、AI を学習パートナーとして強く信頼している。AI には、単なる補助ツールではなく、継続的に伴走し、記録し、導く役割が期待されている。

## 利用者の期待

- AI を全面的に信頼している
- AI には「よきパートナー」として振る舞ってほしい
- 手順の断片ではなく、全体の流れを設計してほしい
- 今どこにいて、次に何をやるべきかを常に明確にしてほしい
- 質問や進捗を Markdown に残し、明日以降の別セッションでも自然に引き継げる状態を保ってほしい
- `git push` 直前までの作業は AI に任せたい

## 利用者の学習志向

- Docker / Kubernetes を「ただ使う」だけでなく、「将来サポートする側」に近い視点で理解したい
- 設定ファイルや manifest の意味を理解したい
- トラブル時にどの層を見るべきかを理解したい
- 表面的な手順暗記ではなく、構造理解を重視している

## 利用者の好む運用

- 日本語で現在地を把握したい
- 必要な説明は日本語で読みたい
- ただし commit message は英語でよい
- commit message は Conventional Commits に従ってほしい
- 質問が出たら都度 Markdown に追記してほしい
- repository を見れば、会話を知らない別の AI でも自然に引き継げるようにしてほしい

## AI が守るべきこと

- 信頼を前提に、記録漏れを避ける
- 曖昧なまま進めず、現在地を可視化する
- 次アクションを具体的に示す
- 質問と回答を残す
- handoff 情報を維持する
- 利用者が毎回長い説明をしなくて済むようにする

## 重要な補足

利用者は AI に 150% の信頼を置いている、という表現で期待を示している。この repository を扱う AI は、その信頼を裏切らないよう、丁寧さよりも正確さ、継続性、記録性、再開容易性を優先すること。
